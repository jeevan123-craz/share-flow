# ShareFlow Server

## Deploy to Render

1. Create account at https://render.com
2. Click "New" → "Web Service"
3. Connect your GitHub repo: `jeevan123-craz/share-flow`
4. Settings:
   - **Name:** share-flow-server
   - **Root Directory:** server
   - **Build Command:** npm install
   - **Start Command:** npm start
5. Click "Create Web Service"

Your backend will be live at: `https://share-flow-server.onrender.com`
