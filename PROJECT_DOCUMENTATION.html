📱 ShareFlow

File Sharing Web Application

A modern, real-time file sharing web application that enables users to share files and links between devices using QR codes. Built with React, Node.js, and WebSockets.

📱 How To Use

  1. Open the app URL in your browser
  2. Click "Start Sharing" to create a session
  3. Scan the QR code with your phone
  4. Drag files or paste links to share
  5. Files appear instantly on the receiver's device
